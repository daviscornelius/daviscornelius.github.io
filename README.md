Running scropplesite locally is easy!
First, you must have NodeJS and NPM installed.

Next, clone the repo.

```
git clone https://github.com/CodeMonster240/Burchweb-Games
```
Open the terminal for the repo, preferably with VSCode's terminal.

Then install the NPM http server:

```
npm i -g http-server
```

**If it says something about a security error, type the following command:**

```
cmd
```
Yeah, that was easy.

Run the http server now:

```
http-server
```

It should open in port `8080`. If it doesn't, that's okay still.
